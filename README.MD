# FlagQuiz 🏴🌍

FlagQuiz é um jogo de perguntas e respostas sobre bandeiras de países, desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Quiz de bandeiras de países
- Dois modos de jogo:
  - Bandeira → Nome do país
  - Nome do país → Bandeira
- Sistema de cadastro e login
- Histórico de partidas
- Ranking semanal
- Perfil de usuário

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Supabase (Autenticação e Banco de Dados)
- GitHub Pages (Hospedagem)
